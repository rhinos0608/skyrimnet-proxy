Configuration Guide

Setting up API Keys

This proxy uses environment variables for API key configuration. Create a .env file in the project root with your provider API keys: 

# OpenAI
OPENAI_API_KEY=sk-proj-your-key-here

# Google AI Studio
GOOGLE_API_KEY=your-google-key-here

# Mistral AI
MISTRAL_API_KEY=your-mistral-key-here

# z.ai
ZAI_API_KEY=your-zai-key-here

# OpenRouter
OPENROUTER_API_KEY=sk-or-your-key-here

# Groq
GROQ_API_KEY=gsk-your-key-here

Provider Base URLs

Each provider is configured with its base URL in config/providers.yaml. The proxy routes requests to these endpoints based on the model slot configuration in config/routes.yaml.

Model Slots

Model slots map friendly names to specific provider/model combinations. Configure these in config/routes.yaml: 

model_slots:
  default:
    provider: zai
    model: GLM-4.7
  fast:
    provider: zai
    model: GLM-4.5-flash
  creative:
    provider: mistral
    model: mistral-large-2411
